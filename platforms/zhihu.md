# 知乎调研指南

## 特点
- 深度分析
- 理性讨论
- "谢邀"、"先说结论"文化
- 数据和案例支撑

## 搜索关键词模板
```
如何看待 <主题>？
<主题> 是一种什么样的体验？
为什么 <主题> 这么火？
有哪些关于 <主题> 的误区？
<主题> 怎么入门？
<主题> 有哪些坑？
```

## 提取要点
1. **高赞回答的开头方式** - "谢邀"、"先说结论"、"实名反对"
2. **专业术语与通俗表达的平衡** - 怎么把专业的话说得人话
3. **数据和案例的使用** - 他们用什么数据、什么案例
4. **知乎特有句式** - "这道题我会"、"刚好在做这个"、"简单说一下"
