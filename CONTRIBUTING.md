# 贡献指南

欢迎为 dankoe-writer 做贡献！🎉

## 贡献方式

### 1. 添加新平台支持

在 `platforms/` 目录下添加新文件：

```markdown
# 平台名称调研指南

## 特点
- 平台特点1
- 平台特点2

## 搜索关键词模板
```
关键词1
关键词2
```

## 提取要点
1. 要点1
2. 要点2
```

### 2. 分享优秀案例

在 `references/examples/` 目录下添加你的优秀案例。

### 3. 改进框架

发现 Dan Koe 框架有可以改进的地方？欢迎提 Issue 或 PR。

### 4. 翻译

欢迎翻译成其他语言！

## 提交 PR

1. Fork 这个仓库
2. 创建你的 feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit 你的 changes (`git commit -m 'Add some AmazingFeature'`)
4. Push 到 branch (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

## 行为准则

- 保持友好和尊重
- 接受不同的观点
- 专注于对社区最好的事情

## 问题？

有问题欢迎提 Issue！
