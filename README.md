Dan Koe 爆款内容生成器 - 基于 The Modern Writing System，整合多平台调研、爆款框架写作、AI 人性化处理于一体。

## 功能

- **多平台调研** - Reddit、小红书、知乎、微博、Twitter、YouTube、抖音、公众号
- **Dan Koe 框架** - Hook → Story → Lesson → Offer 四步爆款法
- **AI Humanizer** - 去除 AI 腔调，写出真人感
- **多格式输出** - 长文、推文线程、小红书笔记、钩子库

## 快速开始

在 Claude Code 中运行：

```
/dankoe-writer 你的主题
```

## 开源协作

欢迎 PR！你可以：

1. 添加新平台支持
2. 分享优秀案例
3. 改进框架和模板
4. 翻译多语言版本
